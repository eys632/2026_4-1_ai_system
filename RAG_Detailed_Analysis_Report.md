| **속도/복잡도 트레이드** | 0.98ms (무시할 수준) |

#### ✨ LLM-Aware RAG의 장점

1. **정확한 의도 파악**
   - "신입생이 배우기 쉬울?" → 신입생 카테고리 자동 인식
   - "어떤 활동?" → 학과활동 카테고리 자동 인식

2. **높은 관련성**
   - 평균 유사도 0.228 (다른 방식의 3~5배)
   - 사용자가 원하는 답변을 더 정확하게 제공

3. **카테고리 기반 필터링**
   ```
   쿼리에서 '졸업', '취업' 감지 
   → 진로취업 문서 우선 검색
   → 더 정확한 답변
   ```

4. **확장 가능성**
   - 새로운 카테고리 힌트 추가 용이
   - 도메인별 최적화 가능

#### ⚠️ 고려사항

- 구현 복잡도: 약간 높음 (하지만 관리 가능)
- 속도 감소: 0.03ms (무시할 수준)
- 유지보수: 카테고리 힌트 업데이트 필요
